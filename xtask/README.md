# `xtask`: Custom Tasks

> `cargo xtask` must be run from the root of the repository.

```sh
# Read help
cargo xtask --help
# Run a task
cargo xtask <task>
```

> **Note**: `cargo xtask` is an alias for `cargo run -p xtask --`. This is defined in the `.cargo/config.toml` file.
